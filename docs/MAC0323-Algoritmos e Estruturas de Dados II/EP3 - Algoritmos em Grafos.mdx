# EP3 - Algoritmos em Grafos

## Descrição do Problema

Este exercício implementa algoritmos avançados em grafos:

1. **Caminhos Mínimos**: Algoritmo de Dijkstra
2. **Ordenação Topológica**: Para grafos acíclicos direcionados
3. **Componentes Fortes**: Algoritmo de Kosaraju ou Tarjan

## Algoritmos Implementados

### Dijkstra
- Encontra caminho mínimo de um vértice para todos os outros
- Usa fila de prioridade (min-heap)
- Complexidade: O((V+E) log V)

### Ordenação Topológica
- Baseada em DFS
- Usada para scheduling de tarefas com dependências

### Componentes Fortemente Conectados
- Decomposição do grafo em SCCs
- Algoritmo de Kosaraju (duas passadas de DFS)
