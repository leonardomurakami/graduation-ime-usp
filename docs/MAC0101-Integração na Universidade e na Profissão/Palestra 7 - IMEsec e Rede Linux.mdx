## Rede Linux

### O que é GNU?

**Richard Stallman:**
- 1971 → MIT Programador
- 1983 → Projeto GNU (GNU is not Unix)
- Criou muitos pacotes de usuário úteis (GCC e GNU Make)
- 1991 → Faltava algo para o projeto continuar, faltava o kernel
- 1991 → Linux Lançado com licença não compatível com o projeto GNU (não permitia comercialização)
- 1992 → Linus trocou a licença do Linux para uma compatível com o GNU
