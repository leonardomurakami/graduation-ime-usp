# EP3 - Sistemas Lineares e Integração

## Descrição do Problema

Este projeto implementa métodos numéricos para:

1. **Resolver sistemas lineares**: Eliminação gaussiana com pivotamento parcial
2. **Integração numérica**: Regras de Simpson e trapezoidal

## Algoritmos Implementados

### Resolução de Sistemas Lineares
- Eliminação gaussiana com pivotamento parcial
- Decomposição LU para múltiplos sistemas
- Cálculo de determinante

### Integração Numérica
- Regra do trapézio composta
- Regra de Simpson 1/3 e 3/8
- Estimativa de erro

## Estrutura do Projeto

- **`ep3.c`**: Implementação principal
- **`ep3.h`**: Cabeçalhos e definições
- **`Makefile`**: Automação de compilação
- **`relatorio.pdf`**: Análise dos resultados
- **`LEIAME`**: Instruções de uso
