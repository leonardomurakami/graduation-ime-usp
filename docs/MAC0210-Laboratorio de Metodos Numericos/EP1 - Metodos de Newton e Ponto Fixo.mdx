# EP1 - Metodos de Newton e Ponto Fixo

## Descricao do Problema

Este projeto implementa dois metodos numericos classicos para encontrar raizes de equacoes:

1. **Metodo de Newton**: Utiliza a derivada da funcao para convergencia quadratica
2. **Metodo do Ponto Fixo**: Baseado em iteracoes sucessivas ate convergencia

## Algoritmos Implementados

### Metodo de Newton
- Requer a funcao e sua derivada
- Convergencia mais rapida quando proximo da raiz
- Implementacao em `newton.c`

### Metodo do Ponto Fixo
- Transforma f(x) = 0 em x = g(x)
- Iteracoes sucessivas ate |x_{n+1} - x_n| < epsilon
- Implementacao em `ponto_fixo.c`

## Estrutura do Projeto

- **`newton.c`**: Implementacao do metodo de Newton
- **`ponto_fixo.c`**: Implementacao do metodo do ponto fixo
- **`relatorio.pdf`**: Analise comparativa dos metodos
