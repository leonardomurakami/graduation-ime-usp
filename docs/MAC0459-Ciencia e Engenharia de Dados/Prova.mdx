## Organização dos Arquivos

Os arquivos fornecidos incluem:
- **Analysis.ipynb (Questão 1)**: Notebook contendo a **análise exploratória** dos dados de acidentes aéreos, incluindo visualizações de tendências temporais, distribuição geográfica dos acidentes, análise de causas frequentes e correlações entre variáveis.
- **create_neo4j_database.ipynb (Questão 1)**: Notebook para **geração do banco de dados** Neo4j, demonstrando a modelagem de dados em grafo, criação de relacionamentos entre aeronaves, ocorrências, fatores contribuintes e localidades.
- **data_models.py**: Definição dos **modelos de dados** utilizados na implementação.
- **neo4j_connector.py**: Código para **conexão e execução de queries** no Neo4j, permitindo consultas complexas sobre relacionamentos entre entidades.

### **Estrutura de Pastas** (segundo `readme.txt`)

Item X > Cenipa > xxxxxx.csv
