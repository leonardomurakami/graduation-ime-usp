# Laboratório de Matemática Aplicada

## Sobre a Disciplina

A disciplina **MAP0131** expõe os alunos a problemas concretos de matemática aplicada cuja modelagem utiliza a bagagem matemática básica de geometria e cálculo, e cuja resolução usa software de manipulação matemática (Mathematica).

### Conteúdo Programático

1. **A descoberta do cálculo com os gregos e o sistema matemático**
   - O número Pi
   - Calculando áreas
   - Euclides vs Arquimedes

2. **As equações algébricas**
   - Histórico: teoria vs praxis
   - Soluções de equações quadráticas e cúbicas usando Mathematica
   - Problemas com soluções exatas e aproximadas

3. **As funções elementares**
   - Gráficos e dependência paramétrica
   - Oscilações
   - Decaimento exponencial

4. **Modelos usando equações de recorrência (sequências)**
   - Juros compostos
   - Crescimento populacional
   - Discretização
   - Modelos econômicos

5. **Máximos e mínimos**
   - Problemas de corte
   - Problemas em grafos (caminho mínimo)
   - Minimização de combustível e tempo mínimo

6. **Cônicas**
   - Problemas de acústica
   - Problemas de ótica (reflexos)
   - Seção cônica

7. **Modelos de interpolação**
   - Modelos topográficos
   - Modelos da prancha de surf

8. **Introdução a modelos dinâmicos**
   - Dinâmica discreta: sistemas de manufatura, problemas populacionais
   - Dinâmica contínua: massa-mola, pêndulo simples

## Programa 1 - Introdução ao Mathematica

Primeiros passos com o software Mathematica para manipulação simbólica e numérica:
- Sintaxe básica e operações
- Resolução de equações
- Geração de gráficos
- Aplicações em cálculo diferencial e integral

## Tecnologias Utilizadas

- **Mathematica**: Software principal de computação simbólica
- **Python**: Alternativa para algumas implementações
- **Jupyter Notebook**: Ambiente interativo de desenvolvimento
