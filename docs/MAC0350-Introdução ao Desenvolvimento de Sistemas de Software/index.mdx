# Introdução ao Desenvolvimento de Sistemas de Software

## Sobre a Disciplina

A disciplina **MAC0350** capacita os alunos a projetar, implementar e testar sistemas de software avançados, usando conceitos e técnicas de engenharia de software e banco de dados de maneira integrada e evolutiva.

### Conteúdo Programático

- **Boas práticas metodológicas** para desenvolvimento de software colaborativo
- **Orientação a objetos**: encapsulamento, herança, polimorfismo, interfaces, tratamento de exceções
- **Técnicas para coleta de requisitos**, modelagem, projeto e implementação de sistemas
- **Técnicas avançadas de teste de software** incluindo arcabouços para automatização de testes
- **Refatoração** e **Padrões de projeto de software**
- **Persistência de dados**: projeto conceitual, lógico e físico de bancos de dados; normalização
- **Modelo entidade relacionamento** e **modelo relacional**: álgebra relacional, SQL

## Aulas Práticas

### Aula 1 - HTML/CSS Básico
Introdução ao desenvolvimento web:
- Estrutura HTML
- Estilização com CSS
- Layouts responsivos básicos

### Aula 2 - JavaScript e Interatividade
- Sintaxe JavaScript
- Manipulação do DOM
- Eventos e interatividade

### Aula 3 - Formulários e Validação
- Criação de formulários HTML
- Validação client-side
- Processamento de dados

### Aula 5 - Flask e Aplicações Web
- Introdução ao framework Flask (Python)
- Rotas e views
- Templates e renderização
- Integração com banco de dados
