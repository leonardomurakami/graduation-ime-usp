# Programa 1 - Busca no Pacman - MAC0425

## Visão Geral

Implementação de algoritmos de busca no ambiente do Pacman, cobrindo busca em profundidade (DFS), busca em largura (BFS), custo uniforme (UCS) e busca A* com heurísticas.

## Algoritmos Implementados

- **Busca em profundidade (DFS)** — busca em grafo com controle de estados visitados
- **Busca em largura (BFS)** — exploração nível a nível
- **Custo uniforme (UCS)** — expansão pelo menor custo de caminho
- **Busca A*** — custo uniforme com função heurística admissível
- **Heurísticas para CornersProblem** — distância Manhattan para cantas não visitadas
- **Heurísticas para FoodSearchProblem** — distância máxima para pontos de comida

## Código-Base

O projeto utiliza o framework do Pacman da UC Berkeley (CS188), adaptado para a disciplina.
