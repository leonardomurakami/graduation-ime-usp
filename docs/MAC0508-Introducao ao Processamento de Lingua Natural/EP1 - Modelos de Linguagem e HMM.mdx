# EP1 - Modelos de Linguagem e HMM

## Parte A: Modelo de Bigramas

Implementação de um modelo de linguagem baseado em bigramas com:
- Cálculo de probabilidades de co-ocorrência
- Geração de texto
- Suavização (smoothing) de probabilidades

## Parte B: Modelo de Markov Oculto (HMM)

Implementação de um HMM para etiquetagem morfossintática:
- Algoritmos Forward-Backward
- Algoritmo de Viterbi para decodificação
- Treinamento com dados anotados
