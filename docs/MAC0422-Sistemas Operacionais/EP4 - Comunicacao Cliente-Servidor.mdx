# EP4: Comunicação Cliente-Servidor

## Descrição do Problema

Este exercício implementa um sistema distribuído cliente-servidor:

- **Sockets**: Comunicação via rede usando TCP/UDP
- **Protocolo**: Definição de mensagens entre cliente e servidor
- **Concorrência**: Múltiplos clientes simultâneos
- **Sincronização**: Coordenação de acesso a recursos compartilhados

## Arquitetura

### Servidor
- Escuta em porta específica
- Aceita múltiplas conexões (fork ou thread por cliente)
- Processa requisições e retorna respostas
- Gerencia estado compartilhado entre clientes

### Cliente
- Conecta ao servidor
- Envia requisições
- Recebe e processa respostas
- Interface de usuário (CLI ou comandos)
