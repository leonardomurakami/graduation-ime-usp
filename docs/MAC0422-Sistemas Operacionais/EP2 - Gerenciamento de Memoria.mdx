# EP2: Gerenciamento de Memória

## Descrição do Problema

Este exercício foca na análise de uso de memória e desempenho de programas:

- **Profiling de memória**: Uso do Valgrind (Massif) para análise de heap
- **Profiling de tempo**: Medição de desempenho de diferentes implementações
- **Otimização**: Identificação de gargalos e melhorias

## Ferramentas Utilizadas

- **Valgrind/Massif**: Análise detalhada de alocação de memória
- **Scripts Python**: Processamento e visualização de resultados
- **Benchmarks**: Comparação de diferentes estratégias

## Estrutura do Projeto

- **`ep2.c`**: Implementação principal para análise
- **`massif_analyzer.py`**: Analisador de logs do Valgrind Massif
- **`time_analyzer.py`**: Analisador de métricas de tempo
- **`run_memory_tests.sh`**: Script para execução de testes de memória
- **`run_time_tests.sh`**: Script para execução de testes de tempo
