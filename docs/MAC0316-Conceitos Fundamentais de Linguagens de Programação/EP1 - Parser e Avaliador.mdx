# EP1 - Parser e Avaliador

## Descrição do Problema

Este exercício implementa um parser e avaliador para uma linguagem simples, incluindo:

1. **Lexing**: Tokenização da entrada
2. **Parsing**: Construção da árvore sintática abstrata (AST)
3. **Avaliação**: Interpretação da AST

## Implementação

### Parser
- Análise sintática recursiva descendente
- Tratamento de expressões aritméticas
- Precedência de operadores

### Avaliador
- Ambiente de execução
- Avaliação de expressões
- Tratamento de variáveis
