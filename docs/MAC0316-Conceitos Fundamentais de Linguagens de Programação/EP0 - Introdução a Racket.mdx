# EP0 - Introdução a Racket

## Descrição

Este exercício introdutório apresenta os fundamentos da linguagem Racket (dialeto Scheme):

- Sintaxe básica e expressões S-expr
- Funções como valores de primeira ordem
- Recursão e processamento de listas
- Ambientes e escopo léxico

## Conteúdo

- Definição de funções simples
- Operações com listas (car, cdr, cons)
- Recursão em cauda (tail recursion)
- Funções anônimas (lambda)
