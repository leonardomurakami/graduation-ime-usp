# Conceitos Fundamentais de Linguagens de Programação

## Sobre a Disciplina

A disciplina **MAC0316** introduz os alunos aos principais paradigmas de linguagens de programação e aos conceitos fundamentais a eles relacionados.

### Conteúdo Programático

- **Linguagens funcionais**: Funções como valores de primeira ordem; polimorfismo; ambientes e fechamentos; avaliação "adiada"
- **Linguagens orientadas a objetos**: Tipos abstratos de dados, módulos, herança e hierarquias
- **Linguagens lógicas**: Predicados, cláusulas e o modelo de unificação
- **Implementação**: Implementação dos vários paradigmas em um interpretador; compilação vs implementação
- **Administração de memória**: Pilha vs heap; coleta de lixo

## Trabalhos Práticos

### Programa 0 - Introdução a Racket
Introdução à linguagem Racket/Scheme, sintaxe básica e conceitos funcionais.

### Programa 1 - Parser e Avaliador
Implementação de um parser e avaliador para uma linguagem simples, manipulando tokens e construindo árvores sintáticas.

### Programa 2 - Interpretador Completo
Desenvolvimento de um interpretador completo com suporte a:
- Variáveis e ambientes
- Funções de primeira classe
- Closure (fechamentos)
- Recursão
- Avaliação lazy (preguiçosa)
