## Projeto - Sistema de Recomendação de Filmes

O projeto final consiste na criação de um sistema baseado em conhecimento para recomendação de filmes, incluindo:

### Componentes
- **Ontologia de filmes**: Representação do domínio usando RDF
- **Base de conhecimento**: Regras sobre filmes, atores, gêneros
- **Motor de inferência**: Sistema de recomendação baseado em preferências
- **Consultas SPARQL**: Para recuperação de informações

### Arquivos do Projeto
- **`ontology-movies.rdf`**: Ontologia em formato RDF
- **`rules.json`**: Regras de inferência e recomendação
- **`sparql_queries.txt`**: Consultas SPARQL de exemplo

## Tecnologias Utilizadas

- **Prolog**: Programação lógica
- **RDF/OWL**: Representação de ontologias
- **SPARQL**: Consultas em bases de conhecimento
- **JSON**: Armazenamento de regras
