Desafios para Casa: JavaScript Básico

Utilize os conceitos vistos na aula para resolver os desafios abaixo. Você deve escrever o HTML e incluir a tag <script> no próprio arquivo por enquanto.

Desafio 1: O Interruptor (Modo Claro/Escuro simples)

Cenário: Crie um botão e um texto que diz "A luz está APAGADA".

Requisitos:

  • Crie uma função chamada alternarLuz() e coloque no onclick do botão.
  • Use document.querySelector para pegar o texto.
  • Use uma condicional (if e ===). Se o texto for "A luz está APAGADA", mude para "A luz está ACESA". Se não, mude de volta para "A luz está APAGADA".
A luz está APAGADA

Desafio 2: O Multiplicador

Cenário: Crie uma variação do contador da aula. Desta vez, vamos multiplicar os números.

Requisitos:

  • Comece com uma variável let valor = 2; e exiba esse número em um <h1>.
  • Crie um botão "Dobrar" com uma função atrelada a ele.
  • Toda vez que clicar, o número deve multiplicar por 2 (ex: valor = valor * 2;) e atualizar a tela.
  • Extra: Usando Template Literals, se o valor passar de 100, exiba um alerta: `Cuidado, o número ${valor} está ficando muito grande!`
